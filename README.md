## 🚀 **Getting Started: From Zero to Hero**

### **Prerequisites**
- Node.js 18+ (We use the latest LTS)
- MongoDB 6+ (For robust data storage)
- Git (Obviously)

### **Installation**

```bash
# Clone the repository
git clone https://github.com/your-org/india-carbon-registry.git
cd india-carbon-registry

# Install frontend dependencies
npm install

# Install backend dependencies
cd backend
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Start the development servers
npm run dev        # Frontend (Vite dev server)
cd backend && npm run dev  # Backend (Nodemon)
```

### **Environment Setup**
```env
# Backend .env
MONGODB_URI=mongodb://localhost:27017/carbon-registry
JWT_SECRET=your-super-secure-jwt-secret
EMAIL_HOST=smtp.gmail.com
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
```
